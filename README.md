## Installation

1. Make a conda environment

```
conda create -n gb python=3.9
conda activate gb
```

2. Install dependencies
```
git clone https://github.com/sunilegna/GB-CGCNN.git
pip install poetry
poetry install
```

3. Training code
```
python main.py
```
